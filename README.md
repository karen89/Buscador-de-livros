# Buscador de Livros Interativo

Este é um projeto de um buscador de livros interativo que permite aos usuários pesquisarem uma coleção de livros por título, autor, categoria ou ano de publicação.

Funcionalidades:

Busca dinâmica em tempo real.
Opções para agrupar resultados por autor ou gênero.
Alternador de tema (claro/escuro) para melhor experiência do usuário.
Exibição da capa do livro em um modal ao clicar.
Design responsivo e amigável.
Tecnologias Utilizadas:

HTML5
CSS3
JavaScript (ES6+)
